📈 Key Insights from the Dashboard
🔹 Analyze sales performance across multiple cities such as Mumbai, Delhi, Chennai, Pune, and more.
🔹 Compare total sales generated by different sales executives.
🔹 Monitor Target Hit % to identify top-performing team members.
🔹 Track Away From Target % to highlight improvement opportunities.
🔹 Use interactive filters to quickly drill down into city-specific performance.
